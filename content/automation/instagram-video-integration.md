# 인스타그램·영상제작 연동

솔티탭 90일 여정은 촬영 원본을 기반으로 릴스와 카드뉴스를 빠르게 제작하고, 승인 후 인스타그램에 게시하는 구조로 운영합니다.

## 자동화 목표

```text
촬영 원본 업로드
↓
개똥이 컷 순서/자막/캡션 생성
↓
Creatomate 영상 렌더링
↓
MP4 검수
↓
승인 후 Instagram Graph API 업로드/게시
```

## 오늘 준비된 연동 패키지

| 항목 | 상태 |
|---|---|
| Instagram Graph API 스크립트 | 준비 완료 |
| Creatomate 렌더링 스크립트 | 준비 완료 |
| Day 001 릴스 캡션/CTA | 준비 완료 |
| Creatomate 템플릿 수정값 샘플 | 준비 완료 |
| 환경변수 템플릿 | 준비 완료 |
| 실제 계정 연결 | 토큰 입력 후 가능 |

## 필요한 계정 정보

### Instagram Graph API

필수 조건:

- Instagram 프로페셔널 계정
- Facebook Page 연결
- Meta Developer App
- Instagram Graph API 권한
- `IG_USER_ID`
- `INSTAGRAM_ACCESS_TOKEN`

### Creatomate

필수 조건:

- Creatomate 계정
- `CREATOMATE_API_KEY`
- 템플릿 사용 시 `CREATOMATE_TEMPLATE_ID`

## 운영 원칙

- 승인 없는 외부 게시 금지
- 게시 전 MP4와 캡션을 반드시 검수
- 효능/치료/전해질 보충 보장 표현 금지
- A타겟은 “땀을 많이 흘리는 날, 물과 함께 챙기는 루틴”으로 표현
- B타겟은 “아침 물 한 잔 옆 10초 소금차 루틴”으로 표현
- 구매 직전 상담은 네이버 톡톡 우선

## Day 001 릴스 구조

| 릴스 | 타겟 | 주제 |
|---|---|---|
| 릴스 1 | A타겟 | 땀을 많이 흘리는 날 물 루틴 |
| 릴스 2 | B타겟 | 아침 물 한 잔 옆 10초 루틴 |
| 릴스 3 | 공통 | 요리용 소금이 아닌 소금차 타블릿 |

## 오늘 밤 촬영 후 진행

촬영 원본이 올라오면 다음 순서로 진행합니다.

```text
1. 원본 파일 분류
2. A타겟/B타겟 컷 매칭
3. 릴스 3개 컷순서 확정
4. 자막/캡션/CTA 확정
5. Creatomate 렌더링
6. MP4 검수
7. 승인 후 인스타 업로드
```
