# 정하은 Automation Bot 보고서

## 역할

정하은은 Obsidian 내부 기록, GitHub Pages 공개 기록, Google Drive 공유, KPI 시트 연결을 관리하는 기록·자동화 담당입니다.

## 공개 보고 기준

- 자동화된 기록 구조
- 공개 사이트 업데이트 내역
- 링크 추적 체계
- 반복업무 개선 사항

## 현재 자동화 구조

```text
내부 Obsidian 기록 → 공개용 요약 변환 → Quartz/GitHub Pages 공개
```
