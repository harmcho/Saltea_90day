# 공개 KPI 대시보드

이 페이지는 솔티탭 90일 여정의 공개 가능한 지표만 정리합니다.  
내부 매출 원자료, 고객 개인정보, 원가·마진, 광고비 세부 내역은 공개하지 않습니다.

## 90일 진행 현황

| 항목 | 현황 |
|---|---|
| 진행 상태 | 준비 / Day 1 시작 대기 |
| 공개 기록 | Day 001 공개 |
| 핵심 구호 | 솔티탭으로 일 천 매출 달성하자 |
| 핵심 실험 | 소금차 타블릿 메시지 검증 |
| 주요 유입 채널 | 인스타그램 → 스마트스토어 / 카카오채널 |
| 구매 확인 링크 | [스마트스토어에서 보기](https://mkt.shopping.naver.com/link/6a26adb52f37317255999973?nt_source=github_pages_kpi&nt_medium=public_journey) |

## 공개 가능 지표

| 지표 | 공개 방식 |
|---|---|
| 릴스 업로드 수 | 숫자 공개 가능 |
| 카드뉴스 업로드 수 | 숫자 공개 가능 |
| 공개 기록 일수 | 숫자 공개 가능 |
| 고객 질문 수 | 범위 또는 요약 공개 |
| 많이 나온 질문 | 개인정보 제거 후 공개 |
| 콘텐츠에서 배운 점 | 공개 가능 |
| 내일의 개선 방향 | 공개 가능 |

## 방향성으로만 공개할 지표

| 지표 | 공개 방식 |
|---|---|
| 조회수 | 전일 대비 증가/감소 중심 |
| 링크 클릭 | 전일 대비 흐름 중심 |
| DM 문의 | 범위 또는 변화 중심 |
| 주문 수 | 내부 관리, 공개 시 흐름 중심 |
| 매출 | 내부 관리, 공개 시 달성률/방향성 중심 |

## 비공개 지표

| 지표 | 비공개 이유 |
|---|---|
| 정확한 일매출 | 경쟁/거래처 리스크 |
| 원가·마진 | 사업 핵심 정보 |
| 광고비 세부 내역 | 경쟁 리스크 |
| 고객 이름/연락처 | 개인정보 |
| DM 원문 | 동의 없는 공개 위험 |
| B2B 거래처명 | 영업 리스크 |
