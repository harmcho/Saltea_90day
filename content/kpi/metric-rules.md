# 지표 공개 기준

솔티탭 90일 여정의 지표는 내부 운영과 외부 공개를 분리합니다.

## 내부용 지표

내부 Obsidian과 Google Sheet에서 상세 관리합니다.

- 정확한 매출
- 주문 수
- 객단가
- 광고비
- ROAS
- 고객별 상담 내용
- B2B 거래처 진행상황
- 재고/원가/마진

## 공개용 지표

GitHub Pages 공개 사이트에는 다음처럼 가공해서 공개합니다.

- 업로드 수
- 공개 기록 일수
- 가장 많이 나온 질문
- 고객 반응에서 배운 점
- 전일 대비 변화
- 이번 주 실험 주제
- 다음 개선 방향

## 공개 문장 예시

좋은 표현:

```text
오늘은 제품 설명형 콘텐츠보다 루틴형 콘텐츠에 반응이 더 좋았습니다.
```

주의할 표현:

```text
오늘 매출이 얼마였고 광고비를 얼마 썼습니다.
```

## 기준 문장

> 지표는 자랑이 아니라 개선을 위한 기록입니다.
